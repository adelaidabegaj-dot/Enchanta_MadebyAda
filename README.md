# Enchanta

A static HTML, CSS, Bootstrap, and vanilla JS website.

## Getting Started

No build step or dependencies are required. Serve the project root with
any static file server and open `index.html`, for example:

```bash
npx serve .
```

## Project Structure

- `index.html` — page markup for all sections
- `css/style.css` — theme variables, keyframes, shared utilities
- `css/sections.css` — per-section styles
- `js/main.js` — nav, sparkles, bubbles, scroll-reveal, carousel, filters, forms
- `assets/` — images and videos
